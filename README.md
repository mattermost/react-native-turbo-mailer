# react-native-turbo-mailer

a react native library to send email using Turbo Module

## Installation

```sh
npm install react-native-turbo-mailer
```

## Usage

```js
import { multiply } from 'react-native-turbo-mailer';

// ...

const result = await multiply(3, 7);
```
